A render node controlling the color matrix of its single child node.

classmethod new(child, color_matrix, color_offset)¶
Parameters:
  • child (Gsk.RenderNode) – The node to draw
  • color_matrix (Graphene.Matrix) – The matrix to apply
  • color_offset (Graphene.Vec4) – Values to add to the color
Returns:

A new GskRenderNode
Return type:

Gsk.ColorMatrixNode

Creates a GskRenderNode that will drawn the child with color_matrix.

In particular, the node will transform the operation

pixel = color_matrix * pixel + color_offset

for every pixel.
